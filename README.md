# vite-h5-template

### 使用
```
# 安装依赖
npm install

# 运行
npm run dev

# 打包
npm run build
```

### 启动mock服务
```
# 打开新的终端窗口，进入到相应的路径
cd api-service

# 安装依赖
npm install

# 启动mock数据接口服务
npm run start
```

### 相关技术栈
JavaScript框架：react
构建工具：vite + rollup
CSS处理：less + postcss-pxtorem
网络请求：axios
状态管理：zustand
UI组件库：antd-mobile
移动端适配：使用宽度为750px的设计稿开发，尺寸单位：直接使用px，会自动转为rem
